Vomit Vortexとは

RimworldのMODを見やすい一覧にして表示することで
あなたのMOD管理をすこしだけ楽にしてくれるソフトウェアです

どのような場面が楽になるのか

  • 「競合を回避する為にパッチ型MODを後にしたい」

    そのような希望が仮にあったとして、
    果たしてあなたはどれがどの型であるか把握できているでしょうか？

    Vomit Vortexを使うことによって
    あなた自身がMODの見分け方を知らなくとも
    MOD毎にどのタイプであるのかを簡単に見られるようになります

  • 「探しているMODがあるけど名前を思い出せない」

    そのような場合でも、Vomit Vortexを使えばプレビュー画像を一覧で見られます
    また、ブラウザを通して一覧を見るので Ctrl + F を使った検索もできます

  • 「スプレッドシートにMOD情報をまとめたい」

    Vomit Vortexによって出力される一覧は、
    コピー＆ペーストに配慮された設計になっています

    コピー時には表の中のUIボタンは省略され、
    サムネイルが不要な時は省くことができます

    また、実際の使い勝手を知る為のサンプルページがあります

利用方法

  1. ダウンロードした圧縮ファイルを任意の場所で展開してください
  2. 展開したフォルダを開いて実行ファイルを起動してください
    ( 起動するまで少し時間がかかります )
  3. 出てきたウィンドウのStartボタンを押し、
    プログレスバーが最大になるまでお待ちください ※1
  4. Startボタンのすぐ右のOpenボタンを押してください
  5. ブラウザであなたの利用しているMODが一覧で表示されます ※2

注釈

  1. SteamでサブスクライブしたMODではなく、ローカルMODの一覧を作りたい場合は
    MOD DirectoryでローカルMODのフォルダパスを入力してください
  2. 「This File was NotFound.」が並んでいる行が一覧にある場合、
    その行は一般的にMODに必要とされるファイルが欠けているフォルダを示すものです
    不要なフォルダである可能性がありますので確認してみてはいかがでしょうか
